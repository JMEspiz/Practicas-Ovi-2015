Java es un lenguaje de programación de propósito general,
 concurrente, orientado a objetos que fue diseñado 
específicamente para tener tan pocas dependencias 
de implementación como fuera posible. Su intención 
es permitir que los desarrolladores de aplicaciones 
escriban el programa una vez y lo ejecuten en cualquier 
dispositivo (conocido en inglés como WORA, o "write once, run anywhere"), 
lo que quiere decir que el código que es ejecutado en una plataforma 
no tiene que ser recompilado para correr en otra. 
Java es, a partir de 2012, uno de los lenguajes de programación 
más populares en uso, particularmente para aplicaciones de cliente-servidor de web,
 con unos 10 millones de usuarios reportados.2 3
